7. FUNCIONES

Unidad 7 FUNCIONES

una función se define como un conjunto de
sentencias que realizan una tarea específica. Su utilidad es enorme,
ya que permite dividir un programa en partes más fáciles de
implementar y, en consecuencia, de mantener. La sintaxis con la que se declara una función es la siguiente:


        function nombreFuncion(parametro1, parametro2) {
            ...
        }

7.1 INVOCACIÓN DE UNA FUNCIÓN

Para invocar una función, se utiliza el nombre de la función seguido de
paréntesis y, opcionalmente, se pasan los valores que sean necesarios para
ejecutar la función.


            nombreFuncion(valor1, valor2);

7.2 EL TIPO DE DATOS FUNCTION

Las funciones son objetos de tipo Function. Esto le puede resultar
extraño, pero en JavaScript todo puede ser modelado como un objeto.


        function factorial(x) {
            ...
        };

El alcance o ámbito de una variable es la parte del programa en la que se puede acceder a ella.
