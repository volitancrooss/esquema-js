4. TIPOS DE DATOS PRIMITIVOS

Unidad 4 TIPOS DE DATOS PRIMITIVOS

Los datos se agrupan en tipos que determinan el
conjunto de valores que lo componen y las operaciones en las que
pueden intervenir (además de la forma en la que se representan
internamente por el ordenador).

  • Boolean.
  • Number.
  • BigInt.
  • String.
  • Null.
  • Undefined.

4.1 BOOLEAN

Como se acaba de indicar, el tipo de datos Boolean está formado
solo por los dos valores, que representan los conceptos de verdadero
(true) y falso (false).

4.2 NUMBER

A diferencia de otros lenguajes de programación que distinguen
entre enteros, con signo, sin signo, decimales de simple o doble
precisión, etc., en JavaScript todos los valores numéricos forman
parte del tipo de datos Number.

4.3 STRING

El tipo de datos String se utiliza para el manejo de textos,
formados por una secuencia de caracteres entre comillas.

 "Hola Mundo"

4.4 NULL Y UNDEFINED

Son diferentes ya que el primero representa un
valor nulo o irrelevante, mientras que el segundo es el que tiene una
variable a la que todavía no se le ha asignado ningún valor.

4.5 CONVERSIÓN ENTRE TIPOS DE DATOS PRIMITIVOS

Los operadores están pensados para trabajar con operandos
del mismo tipo, cuando esto no ocurre JavaScript trata de
convertirlos automáticamente al tipo adecuado.


Si se quiere mantener un mayor control sobre la ejecución del programa,
lo más seguro es forzar de forma explícita dicha conversión con las
siguientes funciones globales:

  • Number(). Convierte un tipo de datos primitivo en un número.
  • String(). Convierte un tipo de datos primitivo en una cadena.
  • Boolean(). Convierte un tipo de datos primitivo en un valor booleano.

4.5.1 Conversión a números

Transforma cualquier tipo de dato primitivo en un número es:

Number("123") //123 
Number("1 23") //NaN 
Number("") //0

La conversión de cadenas a números se hace de forma automática en todas las operaciones aritméticas excepto la suma

"10" - "2" //8
"10" * "2" //20
"10" / "2" //5
Number("10") + Number("2") //12
"10" + "2" //102

A parte de Number, JavaScript ofrece otras funciones de conversión como son:

  • parseInt(). Convierte una cadena en un número entero.
  • parseFloat(). Convierte una cadena en un número decimal.

4.5.2 Conversión a cadenas

La función global que transforma cualquier otro tipo de datos
primitivo en una cadena es:

String(123) //"123"
String(false) //"false"
(123).toString() //"123"

Otros métodos de conversión de números a cadenas que le podrían resultar interesantes son:

  • toFixed(número, decimales). Convierte el número en una cadena, que se redondearía con los decimales indicados opcionalmente como argumento.
  • toExponential(número, decimales). Su comportamiento es similar al del método anterior, excepto que el número se expresaría en notación exponencial.
  • toPrecision(número, dígitos). Su comportamiento vuelve a ser similar al del primer método, solo que, en este, el argumento opcional indicaría el número de dígitos al que se redondearía el número.

4.5.3 Conversión a valores booleanos

Solo faltaría saber cómo convertirlo en
true o false. Para ello deberá utilizar la función global:

Boolean(null) //false
Boolean(123) //true
Boolean(0) //false
